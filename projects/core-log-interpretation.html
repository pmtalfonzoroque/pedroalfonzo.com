Phoenix Deposit – Core Logging Interpretation

The Phoenix uranium deposit, located within the Wheeler River Project in Saskatchewan’s Athabasca Basin, displays a well-defined sequence of sandstone, clay-altered zones, friable mineralized intervals, and weathered basement. The core photos below represent the key hydrogeological units encountered in the deposit and form the basis for this interpretation.

Geological Context

Phoenix is a classic unconformity-related uranium system. The sequence transitions from Athabasca sandstone into progressively stronger clay alteration near the unconformity, followed by friable high-grade zones and finally into paleoweathered and competent basement rocks. These units reflect fluid flow pathways, redox reactions, and structural preparation that controlled uranium deposition.

Interpretation of Hydrogeological Units

1. Sandstone Sequence (Units 1a–1b)

The upper sandstone units show partial desilicification, faulting, and localized sulfide cementation. These features indicate fluid movement through brittle structures and early chemical alteration prior to major mineralizing events.

2. Clay and Mineralized Zones (Units 2a–2e)

This interval marks the transition into the most hydrogeologically active part of the system. The upper clay zone (2a) shows strong alteration and reduced competency. The high-grade friable zone (2b) is characterized by honeycomb textures, intense clay alteration, and high porosity—features commonly associated with uranium-rich fluid pathways. The brown redox zone (2c) represents an oxidation boundary, while the sulfide-cemented interval (2d) reflects reducing conditions. The lower clay zone (2e) forms the transition into basement rocks.

3. Basement Sequence (Units 3a–3b)

The paleoweathered basement (3a) shows softened textures and increased porosity, indicating deep weathering prior to burial. Beneath it, the competent basement (3b) marks the return to unaltered crystalline rock. These units are critical for understanding structural preparation and the geometry of mineralizing fluids at the unconformity.

Summary

Together, these units illustrate the vertical evolution of alteration, permeability, and redox conditions that control uranium deposition at Phoenix. The combination of friable textures, clay alteration, and redox contrasts provides a strong geological framework for interpreting core images from any unconformity-related uranium system.

Core‑Image Interpretation Workflow

This project is not a static description of the Phoenix deposit. It is a demonstration of how raw core photographs can be transformed into structured geological interpretation using a reproducible workflow. The goal is to show how engineering thinking, geoscience knowledge, and coding-based automation can work together to support consistent core logging.

1. Image Acquisition & Pre‑Processing

  • Import raw core photos (JPG, PNG, or field images).
  • Standardize orientation and cropping to isolate the core box.
  • Normalize brightness and color to reduce lighting bias.
  • Apply optional filters to enhance textures (clay, friability, sulfides).

2. Feature Identification

  • Detect color domains (brown/red oxidation, grey/green reduction).
  • Identify texture changes (friable zones, honeycomb patterns, clay-rich intervals).
  • Highlight structural features (faults, fractures, broken core).
  • Locate high-contrast mineral patches (sulfides, hematite, pitchblende).

3. Unit Classification Logic

Each hydrogeological unit at Phoenix has diagnostic visual cues. A rule-based or ML-based classifier can suggest the most likely unit based on:

  • Color: red/brown for oxidation (2c), dark patches for sulfides (2d).
  • Texture: friable/honeycomb for high-grade zones (2b).
  • Competency: soft clay vs. competent basement.
  • Structural context: faulted sandstone vs. intact basement.

4. Interpretation Output

The workflow produces a structured interpretation that includes:

  • Suggested unit (1a, 2b, 3a, etc.).
  • Key visual indicators supporting the classification.
  • Confidence level (low/medium/high).
  • Optional: extracted features for ML training.

5. Reproducibility & Automation

The workflow is designed so that the same steps can be applied to any core photo. This supports:

  • Consistent logging across drillholes.
  • Automated pre-screening of mineralized intervals.
  • Training datasets for machine learning models.
  • Rapid interpretation for exploration or QA workflows.
